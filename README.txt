# Análise de Conversas com Meu Filho

Projeto pessoal para analisar e estudar estatisticamente as mensagens trocadas
entre eu e meu filho. Este projeto faz parte de um trabalho maior
relacionado ao meu Projeto de Futuro.

Etapas previstas:
- organizar dados de conversas (texto + áudio)
- aplicar ferramentas estatísticas
- gerar gráficos e insights ao longo do tempo
- documentar tudo neste repositório
